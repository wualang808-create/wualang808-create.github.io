字符串的常用属性

  • 所有对象的调用 用“.”

    • 变量名.方法()

  • 分类

    • 判断类：一般是是is开头，返回值是true / false

    • 分类 方法 详细说明 示例
      数字/字符构成 isdigit() 是否只包含数字字符 (0-9) "123".isdigit() -> True
      isnumeric() 是否为数值 (含中文数字、罗马数字) "五".isnumeric() -> True
      isdecimal() 是否为十进制数字 (最严格) "123".isdecimal() -> True
      isalpha() 是否全是字母 (包含中文/韩文等) "Python".isalpha() -> True
      isalnum() 是否全是字母或数字 (不含空格/符号) "a1".isalnum() -> True
      大小写与格式 islower() 字母是否全为小写 "abc".islower() -> True
      isupper() 字母是否全为大写 "ABC".isupper() -> True
      istitle() 是否为标题格式 (单词首字母大写) "Hello World".istitle() -> True
      isspace() 是否只包含空白字符 (\n, \t, 空格) "\n ".isspace() -> True
      位置与包含 startswith(s) 是否以子串 s 开头 "abc".startswith("a") -> True
      endswith(s) 是否以子串 s 结尾 "abc".endswith("c") -> True
      in (运算符) 字符串是否包含子串 "a" in "abc" -> True
      特殊判断 isidentifier() 是否符合 Python 变量命名规则 "var_1".isidentifier() -> True
      isprintable() 是否全是可打印字符 (不含 \n 等) "abc".isprintable() -> True
      isascii() 是否仅由 ASCII 字符 组成 "abc".isascii() -> True

  • 标识符认证（identifier）：变量名 ：必须是数字、字母、下划线组成，不能以数字开头，关键字不能作为变量名

  • 这类方法的核心逻辑是：检查字符串的特定部分（头部或尾部）是否符合预期的模式。

    位置与匹配类方法总结

    这类方法不仅支持简单的字符串查找，还支持指定范围和多条件匹配。

  • 方法 语法 作用 返回值
    startswith s.startswith(prefix[, start[, end]]) 检查字符串是否以指定子串开头 bool
    endswith s.endswith(suffix[, start[, end]]) 检查字符串是否以指定子串结尾 bool

  • 常用的统计的属性

  • 维度 方法 语法 作用 备注
    整体测量 len() len(str) 计算字符串的总长度（字符个数） 它是内置函数，不是 str 的方法
    内容计数 count() s.count(sub[, start[, end]]) 统计子串 sub 出现的次数 支持指定范围，不重叠计数
    位置检索 find() s.find(sub[, start[, end]]) 查找子串第一次出现的下标 找不到返回 -1 (推荐)
    index() s.index(sub[, start[, end]]) 查找子串第一次出现的下标 找不到会报错 (容易导致程序崩溃)
    rfind() s.rfind(sub) 从右侧开始查找第一次出现的下标 返回该字符在原串中的正向索引

💡 核心知识点补充：

  • 区分大小写：所有统计方法都是区分大小写的。如果你想统计不区分大小写的字符，通常先用 .lower() 转换后再统计。
  • 不重叠统计：count() 在统计时是“不重叠”的。例如 "aaaa".count("aa") 的结果是 2，而不是 3。

  • 转换属性

  • 分类 方法 作用 示例
    大小写变换 upper() 全部转为大写 "abc".upper() -> "ABC"
    lower() 全部转为小写 "ABC".lower() -> "abc"
    capitalize() 首字母大写，其余小写 "hello".capitalize() -> "Hello"
    title() 每个单词首字母大写 "hi world".title() -> "Hi World"
    swapcase() 大小写互换 "AbC".swapcase() -> "aBc"
    内容处理 replace(old, new) 替换指定内容 "abc".replace("a", "1") -> "1bc"
    strip(chars) 去除首尾指定字符（默认空格） " #a# ".strip() -> "#a#"
    lstrip() / rstrip() 只去除左侧或右侧字符 " aa ".rstrip() -> " aa"
    结构变换 split(sep) 按指定分隔符切开，返回列表 "a:b:c".split(":") -> ['a', 'b', 'c']
    join(iterable) 将容器里的元素用该字符串拼接 "-".join(['a', 'b']) -> "a-b"

1. 文本预处理：strip() 与 replace()

这两个方法是处理原始数据（如用户输入、网页爬虫数据）的“神兵利器”：

  • strip()：主要用于清理用户不小心多打的空格或换行符。
  • replace()：可以链式调用。例如 s.replace(" ", "").replace("\n", "") 可以一次性删掉所有空格和换行。

2. 格式规范化：lower() 与 casefold()

  • 在进行忽略大小写的比较时，通常先调用 .lower()。
  • 进阶点：如果你处理的是包含德语（如 ß）等特殊字符的国际化文本，使用 casefold() 比 lower() 更彻底。

3. 数据转换的核心：split() 与 join()

这对方法通常成对出现，是“字符串”与“列表”之间转化的桥梁：

  • split()：将一个长字符串拆解成零件。
  • join()：将一堆零件组装回字符串。注意语法："分隔符".join(列表)，分隔符在前面。

💡 核心提醒：字符串不可变性

这是初学者最容易犯错的地方：

在 Python 中，填充类方法主要用于控制字符串的显示宽度和对齐方式。它们在制作命令行表格、打印菜单或格式化输出日志时非常有用。

这类方法的核心逻辑是：设定一个目标总长度，如果原字符串不够长，就用指定的字符（默认为空格）填充剩余空间。

字符串填充类方法汇总

方法 语法 对齐方式 填充方向
center() s.center(width[, fillchar]) 居中对齐 左右两侧同时填充
ljust() s.ljust(width[, fillchar]) 左对齐 在右侧填充内容
rjust() s.rjust(width[, fillchar]) 右对齐 在左侧填充内容
zfill() s.zfill(width) 右对齐 在左侧填充 0 (常用于编号)

1. 三种对齐方式对比

假设我们有一个字符串 s = "Hello"，我们希望它的总长度达到 10：

  • s.center(10, "-") → "---Hello--"（居中，两边补横杠）
  • s.ljust(10, "\*") → "Hello*****"（左对齐，右边补星号）
  • s.rjust(10, ".") → ".....Hello"（右对齐，左边补点号）

2. 特殊补充：zfill()

zfill() 是 zero fill 的缩写。它不需要指定填充字符，固定填充 0。它能智能处理正负号，非常适合格式化流水号或时间日期。

Python

💡 核心注意事项

  1. 填充字符的限制：fillchar 参数必须是单个字符（长度为 1）。传入 "**" 会报错。
  2. 原串超长：如果设定的 width 小于或等于原字符串的实际长度，Python 不会截断字符串，而是原样返回。
    • "Python".center(3) 依然返回 "Python"。
  3. 默认值：如果不写第二个参数，它们默认都使用 空格 填充。
